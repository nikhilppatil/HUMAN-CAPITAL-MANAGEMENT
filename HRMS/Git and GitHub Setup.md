**Git and GitHub Setup**



1. Create GitHub Account
2. Create Repository on GitHub
3. Install Git on Your System
4. Open Command Prompt / Terminal
5. Navigate to Your Project Folder
   cd path/to/your/folder
6. Initialize Git in the Folder
   git init
7. Add Files to Git
   git add .
8. Check Git Status
   git status
9. Make Your First Commit
   git commit -m "Initial commit: Add SQL folder"
10. Link Local Repo to GitHub Remote Repo
    git remote add origin https://github.com/your-username/repo-name.git
11. Push Code to GitHub

    git push -u origin main

    git push --set-upstream origin master



**For creation of New Branch**



12\. git checkout -b feature/add-sql-script

    git push origin feature/add-sql-script
