**1) User Story: Payment Gateway Configuration**



**Title: Payment Gateway Integration for Hostel Admission Payments**



As a User,

I want to pay my hostel admission fees through various online payment methods configured via the CC Avenue payment gateway,

so that I can complete my admission process quickly and securely using my preferred payment option.



**Description**



Students applying for hostel admission should be able to make online payments through multiple options such as UPI, Net Banking, Credit/Debit Card, and other payment methods supported by CC Avenue.

The system should record payment details, confirm the transaction, and update the admission status automatically.



**Flow of Process**



* User clicks the Payment button



* Redirected to the Payment Gateway



* Payment options displayed (UPI, Net Banking, etc.)



* User selects an option



* User enters required details



* User confirms payment



* System shows transaction status (Success / Failed / Pending)



* On Success, user receives receipt
